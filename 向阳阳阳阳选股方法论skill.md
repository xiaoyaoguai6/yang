- **只买主线的票，其他板块一律不碰**：「除了主线的票全部统统砍掉」（0529 语音确认）；「板块只看光，因为我一直强调我只看光」（0602 05:00 语音确认）

### 2.2 个股筛选标准

#### 三步选股法（大盘→板块→个股）
> 「看盘先看大盘，再看板块，最后最后才是个股。不要搞反了。无源之水，无本之木。」（0601 17:00 语音确认）

#### 技术面要求

**标准图形定义**：一波上涨 → 缩量回调 → 起跳点（放量站上短期均线）

具体条件：
1. **必须有一波明显的上涨行情**（证明有资金看好）
2. **回调过程必须缩量**：「回调时必须缩量，放量就有问题，说明有人出货」（0526 09:00 语音确认）
3. **回调到短期均线附近获得支撑**
4. **放量阳线站上短期均线** = 起跳信号
5. **量价关系健康**：阳量多、阴量小（0604 06:00 语音确认）

**量价四要素（量价时空）**：
> 「其实它正确的解读不是量价关系，是成交量、价格、时间、空间。准确来讲是叫量价时空。」（0604 24:00 语音确认）

**核心量价规律**：
- 上涨必须放量，下跌必须缩量（0611 01:30 语音确认）
- 缩量急跌不可怕，放量下跌才可怕（0629 08:50 语音确认）
- 高位缩量上涨 → 没人跟 → 必然回调
- 低位缩量下跌 → 没人卖 → 变盘信号（0611 02:00 语音确认）

**不选的股票**：
- 下跌量太大、阶梯量不好看的不选
- 有长上影线（冲高回落）的不选
- 量价关系不完美的不选
- 连续上涨不断新高的不选（追高风险大）
> 「没必要做那种没那么好看的，要做就做那种非常完美的。」（0604 03:30 语音确认）

#### 基本面要求

**盘子大小偏好**：
- **优选大盘子**（千亿级）：走势清晰，不会被随意操作
- 工业富联（14000亿）是他反复推崇的标杆：「盘子够大，而且很清晰」（0526 03:30 语音确认）
- 小盘股（几十亿）：波动太大，容易被单一主力操控，「持股体验非常差」（0609 05:30 语音确认）
- 日成交额过百亿的巨盘股（如长电科技）：操作难度极大，不碰（0617 03:30 语音确认）

**「大哥」理论——看主力资金**：
- 「大哥」= 主力资金/大基金
- 判断标准：成交量是否持续活跃、价格是否守住关键位
- **「大哥在 → 拿住；大哥走 → 必走」**
- 用西游记比喻：「大师兄和二师兄都跑了，那你还要在这里搞什么？你要沙僧一个人带着唐僧去取经吗？」（0611 15:30 语音确认）

## 三、买卖时机

### 3.1 买入信号

#### 大盘级别入场信号

**确认性反转信号（最安全）**：
> 「如果大盘要反转，它会有一个明确的信号。第一，大阳线。第二，放量。第三，站上短期均线。」（0612 00:00 语音确认）

- 4月8号的放量大阳线就是标准信号：「要这种放量大阳，才是非常确定的进场信号。你进去之后就没有什么风险，就可以放心大胆地干」（0609 03:30 语音确认）

**7月大盘关键位**（0701 语音确认）：
- 大盘冲高回落，需站上4120才能维持上涨趋势
- 不够强势则可能继续磨，但跟科技股没有关系，是整体市场环境

**底部区间四维确认法**（0611 语音确认）：
1. **K线**：出现止跌K线
2. **量能**：成交量持续萎缩 → 「缩到底了就差不多了」
3. **时间**：「跌的时间够长了」
4. **空间**：「跌去了三分之二……空间也够了」

**反弹 vs 反转的区分**：
> 「你这里你得明确进去一定是做反弹的，而不是做反转。要出现这种K线，才是反转。」（0609 03:30 语音确认）
- 做反弹：仓位轻、预期低、随时准备走
- 做反转：确认信号出来后可以重仓

#### 个股级别买入信号

**标准买入流程**（以北方华创为例，0526 08:00 语音确认）：
1. 前面经历一轮下跌
2. 某天出现放量（资金进场信号），但**这天不能买**（转折点≠买点）
3. 等它上涨，上涨后再等它回调
4. 回调后看有没有承接（有没有资金来接盘）
5. 回调到短期均线附近，放量站上短期均线 → **选出**这个票
6. 再等一天确认 → **进场**
7. 进场后如果缩量回调但不破均线 → 放心拿着

**「起跳点」成立条件**（0526 09:00 语音确认）：
- 买入后不能跌破短期均线
- 必须缩量回调，放量就有问题（说明有人出货）
- 「这种起跳点就是成立的，他买入第二天不涨无所谓，只要他不跌破短期均线」

**最佳买入时机**：
1. **大跌日 = 调仓换股机会**：「从这天大跌开始，这是调仓换股的机会」（0625 02:20 语音确认）

### 6.1 空仓条件

**必须空仓的信号**：
1. 大盘跌破长期均线 → 高风险，不做反弹（0602 17:30 语音确认）

3. 盘面稳定性差、赌性太强时
4. 没有十足把握时

**空仓时做什么**：
- 把前期看好但一直在涨的股票加入自选，等大盘下跌到位后从中精选
> 「现在空仓了，到底要干点什么事情呢？可以先去找股票了。就比如你之前看好的股票，然后它一直在涨，一直在涨不能买对吧？现在就可以把它们全部拿出来，全部加到自选股里面。」（0601 07:30 语音确认）

**空仓的历史验证**：
- 3月份空仓整整一个月，4月8号放量大阳线才进场，完美规避下跌（0608 00:30 语音确认）

### 6.5 大盘分析框架

**推球原理**（0605 01:00 语音确认）：
> 「一块石头从上面掉下来，它会怎么样？如果你在这里设置一个关卡，如果小石头落到这裡，它是不是就会跳起来，要继续往下。」

分析维度：
1. **下跌力度**：看K线的实体大小和成交量
2. **下跌速度**：时间长短
3. **支撑关卡**：均线位置、缺口位置
4. **反弹力度**：反弹是否放量、是否站上均线

**早盘3分钟判断法**（0608 01:00 语音确认）：
1. 大幅低开 → 表明意图是要「杀」
2. 开盘后往上拉 → 不是真正的反弹，是为了杀更多人
3. 先杀恐慌盘（量化止损+散户恐慌卖出）
4. 再杀抄底盘（拉起来让人以为是机会，然后反杀）

**TCL科技**（6月）：
- 「横有多长竖有多高」，底部横盘几年
- 9号开始进场，一直拿得很稳
- 上涨→缩量回调→上涨的健康节奏
- 教训：有耐心就能等到翻倍

**光迅科技**（6月）：
- 大哥跑了后缩量、价格不破
- 缩量站住→涨停，30+个点收益后开始减仓
- 教训：大哥走了但价格不破，说明有新资金进场

**5月6日策略切换**：
- 彻底砍掉非主线仓位（比亚迪、中国卫星等）
- 全部转入光通信主线
- 结果：5月大丰收
- 教训：主线切换时要果断，不要恋旧

### 7.2 失败/教训案例

**6月月度总结**（0701 语音确认）：
- 月初空仓亏3个点，整月盈利18个点
- 做对的：压对TCL+风华高科（玻璃基板+MLCC），进场点判断正确
- 做错的：月尾三天连续跌停不在预案内，科技股内部严重分化
- 教训：「从结果来看它是错的，但从逻辑来看它是对的。我依旧会选逻辑对的那个，这是胜率问题」

**工业富联走弱换TCL**（0701 语音确认）：
- 工业富联没有继续新高，板块新高个股不新高 → 不喜欢了
- 转做TCL：进入拉升阶段，缩量涨停=启动标志
- 教训：同一板块内要跟着强的走，不要恋旧

**四方股份**（0701 语音确认）：
- 图形非常好看，缩量回调后放量往上冲，但没有成功
- 「该涨不涨很难受」，跌破短期均线后清仓
- 教训：强势票的调整方式不同——不往下打价格，而是横盘小幅回调

**东山精密**（5-6月）：
- 高位震荡一直涨，后因墨迹太久+大盘风险0526早盘卖出，平出
- 后出现跌停放量典型出货信号
- 教训：涨不动就走，不要等

**永鼎股份**（5月）：
- 0525说不拉就走，0526跌破短期均线早盘卖出
- 后0527买回（洗盘确认），0529仍在持仓但表现一般
- 教训：洗盘和出货的判断需要经验，第一次卖早了

**通鼎互联**（6月）：
- 小盘股，「持股体验非常差……涨一天然后来根出货K线……你就看着就烦」
- 教训：尽量不要做小盘股，钱赚得不舒服不踏实

**双星新材**（6月）：
- 大跌日进场，但后来破位
- 「今天是不该跌的」，缩量急跌16个点
- 教训：小盘股即使图形好也可能被狙击

### 7.3 核心经验教训

1. **主线切换要果断**：5月6日砍掉非主线全进光通信，效果显著
2. **出货K线宁可信其有**：「你一律把它当成真出货，你就一定不会亏钱」
3. **大盘蓝筹比小盘股安全**：工业富联 vs 通鼎互联的对比
4. **洗盘和出货的区分**：等1-2天确认，结合板块和ETF判断
5. **涨不动就走**：阳线缩小、量价背离都是信号
6. **大跌是机会不是风险**：「不要怕下跌……跌得越多，后面赚得越多」
7. **强势票的调整方式**：不是价格往下打，而是横盘小幅回调（0701新增）
8. **科技股拿住别动**：缩量下跌不能卖，放量上涨也不能卖，「涨就是跌，跌就是涨」（0701新增）
9. **板块内部分化要观察**：同板块大票小票走势可能不同，没有标准答案，边走边看（0701新增）
10. **龙头跌下来是机会**：「好不容易龙头能跌下来」，考虑换到龙头上（0701新增）
